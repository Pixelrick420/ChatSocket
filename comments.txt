socket() takes 3 params
    1. IP version(AF_INET = ipv4)
    2. Type (tcp or udp) (SOCK_STREAM = tcp)
    3. Protocol Underneath (0 = IP)

    returns int socket file descriptor
    if failed, returns -1

htos() takes 1 param
    1. ip port number 
    returns 2 byte short int representing that number in big endian (expected by tcp)

inet_pton() takes 3 params
    1. ip version (AF_INET = ipv)
    2. ip adress in string form (ex : "121.21.32.3")
    3. a pointer to 4 byte int where converted ip will be saved. 

connect() takes 3 params
    1. socket file descriptor
    2. pointer to sockaddr structure (sockaddr_in = ipv4)
    3. sizeof that address struct
    returns

send() takes 4 params
    1. Socket file descriptor
    2. message string(char*)
    3. length of message
    4. flags

recv() takes 4 params
    1. socket file descriptor
    2. output buffer
    3. length of output buffer
    4. flags
 
sockaddr_in contains 
    1. int sin_port (port number of server that is listening on the other side)
    2. sin_family (AF_INET = ipv4)
    3. sin_addr.s_addr (ip address 4 bytes)

bind() takes 3 params
    1. server socketFD
    2. address of the server socket 
    3. size of the address

listen() takes 2 params
    1. client socketFD
    2. backlog count (max no of clients at one time)
    
accept() takes 3 params
    1. socketFD of client
    2. client socket address
    3. pointer to size of client socket address
